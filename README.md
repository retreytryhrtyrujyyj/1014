# 户籍管理系统

## 项目简介
基于Java Web的户籍管理系统。包含管理员登录、普通用户登录、新增户籍、管理户籍、删除户籍、户籍迁入、户籍迁出等功能。包含：1、源代码 数据库 2、配套论文 3、答辩PPT 4、一键启动项目 5、远程调试服务

## 技术介绍
Java web、Mysql、JSP/Servlet

Eclipse或者MyEclipse或者IDEA
